|조회수:

:rotating-light:CSS 속성을 정리한 자료입니다.

CSS 배치

HTML 엘리먼트의 위치를 설정하는 속성이다.

position

요소의 위치 지정 방법의 기준을 설정한다.

  • static: 기본값. 배치 불가능
  • relative : 요소 자신을 기준으로 배치
  • absolute : 위치 상 부모 요소를 기준으로 배치
  • fixed : 브라우저를 기준으로 배치
  • sticy : 스크롤 영역 기준으로 배치
