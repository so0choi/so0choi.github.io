pm2лЎң н”„лЎңм„ёмҠӨлҘј мӢңмһ‘н•ҳл Өкі  н–ҲлҠ”лҚ° permission denied м—җлҹ¬к°Җ л°ңмғқн–ҲлӢӨ.

sudo -i лҘј мӮ¬мҡ©н•ҙ root мӮ¬мҡ©мһҗлЎң мӮ¬мҡ©н•ҳкі  мһҲм—ҲлӢӨ. к¶Ңн•ң л¬ём ңмқё кІғ к°ҷм•„ rootк°Җ м•„лӢҢ лӢӨлҘё мӮ¬мҡ©мһҗлЎң н”„лЎңм„ёмҠӨлҘј мӢңмһ‘н•ҳл Өкі  н–ҲлҠ”лҚ° м•„лһҳмҷҖ к°ҷмқҖ м—җлҹ¬к°Җ лӮҳнғҖлӮ¬лӢӨ.

м„ңм№ҳн•ҙліҙлӢҲ root мӮ¬мҡ©мһҗк°Җ м•„лӢҢ мӮ¬мҡ©мһҗлҠ” 1024 мқҙн•ҳ нҸ¬нҠёлҘј м—ҙ мҲҳ м—ҶлӢӨкі  н•ңлӢӨ. (м°ёкі )

н”„лЎңм„ёмҠӨлҠ” root мӮ¬мҡ©мһҗлЎң мӢңмһ‘н•ҙм•јлҗҳлҠ” кІҢ л§һлӮҳліҙлӢӨ..
к·ёлҰ¬кі  кё°лЎқн•ҙл‘җм§Җ лӘ»н–ҲлҠ”лҚ° мқҙкІғм ҖкІғ н•ҙліҙлӢӨк°Җ pm2к°Җ node лІ„м „мқҙ лӮ®лӢӨлҠ” м—җлҹ¬лҸ„ ліҙм—¬мӨ¬лӢӨ. нҷ•мқён•ҙліҙлӢҲ sudo мқј л•ҢмҷҖ sudoлҘј мӮ¬мҡ©н•ҳм§Җ м•Ҡмқ„ л•Ң мӮ¬мҡ©н•ҳлҠ” nodeк°Җ лӢ¬лһҗлӢӨ.

м—җлҹ¬лҘј м°ҫм•„ліҙлӢҲ мқҙ кёҖмқ„ м°ҫм•ҳлӢӨ. (canвҖҷt use nvm from root or sudo) м ңмқј мғҒлӢЁмқҳ лӢөліҖм—җ мһҲлҠ” м»Өл§Ёл“ңлҘј мӢӨн–үн•ҳлӢҲ н•ҙкІ°мқҙ лҗҳм—ҲлӢӨ.

мһҳлӘ»н•ҳл©ҙ мӢңмҠӨн…ңмқ„ л§қк°ҖлңЁлҰҙ мҲҳлҸ„ мһҲлӢӨкі  мЎ°мӢ¬н•ҙм„ң мӮ¬мҡ©н•ҳлқјкі  н•ңлӢӨ.

мқҙм ң sudoлӮҳ rootм—җм„ңлҸ„ nvmмқҙ м„Өм№ҳн•ң л…ёл“ңлҘј мӮ¬мҡ©н•ҳлҠ”лҚ° м—¬м „нһҲ pm2 к¶Ңн•ң л¬ём ңк°Җ н•ҙкІ°лҗҳм§Җ м•Ҡм•ҳлӢӨвҖҰ
к¶Ңн•ң м—җлҹ¬к°Җ лӮң нҢҢмқјл“Өмқ„ chmod н•ҳлҠ”кұҙ мқҳлҜёк°Җ м—ҶлҠ” кІғ к°ҷм•ҳкі  рҹҘІ.. лӢӨлҘё кёҖмқ„ ліҙлӢҲ pm2лҘј мһ¬м„Өм№ҳ н•ҙліҙлқјкі  н•ңлӢӨ.

мқҙл ҮкІҢ н–ҲлҠ”лҚ°лҸ„ м•Ҳлҗҳл©ҙ nodeлҘј мһ¬м„Өм№ҳ н•ҳлқјкі  н•ңлӢӨ! nvmлҸ„ мһ¬м„Өм№ҳ н–ҲлӢӨ.
н•ҳм§Җл§Ң м•„л¬ҙлҰ¬н•ҙлҸ„ к°ҷмқҖ м—җлҹ¬к°Җ л°ңмғқн–ҲлӢӨ. мқјлӢЁ мӢӨн–үмӢңнӮ¬ мҲҳ мһҲлҠ” л°©лІ•мқҖ npm startк°Җ м•„лӢҲлқј sudoлҘј л¶ҷм—¬м„ң м§Ғм ‘ pm2лҘј мӢӨн–үмӢңнӮӨлҠ” кІғмқҙлӢӨ. ..
