GraphQL мқҙлһҖ

мһ‘м„ұмқј2021-01-14|мҲҳм •мқј2022-08-16|GraphQL
|мЎ°нҡҢмҲҳ:

GraphQLмқҖ нҺҳмқҙмҠӨл¶Ғм—җм„ң л§Ңл“  мҝјлҰ¬м–ём–ҙмқҙлӢӨ. л“ұмһҘн•ңм§Җ м–јл§Ҳ м•ҲлҗҗмқҢм—җлҸ„ л¶Ҳкө¬н•ҳкі  мқёкё°к°Җ л§Өмҡ° к°ҖнҢҢлҘҙкІҢ мҳ¬лқјк°Җкі  мһҲм§Җл§Ң м•„м§Ғ GraphQLмқ„ мқҙмҡ©н•ң open APIлҘј м ңкіөн•ҳлҠ” кіімқҖ көӯлӮҙ л°Ҹ н•ҙмҷёлҘј нҸ¬н•Ён•ҙлҸ„ л§Һм§Җм•ҠлӢӨ.

мҝјлҰ¬м–ём–ҙ мҰү, SQLкіј л§Ҳм°¬к°Җм§ҖмқҙлӢӨ. н•ҳм§Җл§Ң л‘ҳмқҳ кө¬мЎ°м Ғ м°ЁмқҙлҠ” л¶„лӘ…н•ҳлӢӨ. SQLмқҙ DB мӢңмҠӨн…ңм—җ м ҖмһҘлҗң лҚ°мқҙн„°лҘј нҡЁмңЁм ҒмңјлЎң к°Җм ёмҳӨлҠ” лӘ©м ҒмңјлЎң л§Ңл“Өм–ҙм§„ л°ҳл©ҙ, GQLмқҖ web clientк°Җ лҚ°мқҙн„°лҘј м„ңлІ„лЎңл¶Җн„° нҡЁмңЁм ҒмңјлЎң к°Җм ёмҳӨлҠ” кІғмқ„ лӘ©м ҒмңјлЎң н•ңлӢӨ. л”°лқјм„ң SQLмқҖ л°ұм—”л“ң мӢңмҠӨн…ңм—җм„ң мӮ¬мҡ©лҗҳлҠ” л°ҳл©ҙ, GQLмқҖ мЈјлЎң нҒҙлқјмқҙм–ёнҠё мӢңмҠӨн…ңм—җ нҳём¶ңлҗңлӢӨ.

GQLмқҖ м–ҙлҠҗ нҠ№м • DBлӮҳ н”Ңлһ«нҸјм—җ мў…мҶҚм Ғмқҙм§Җ м•ҠлӢӨ. мқјл°ҳм ҒмңјлЎң GQLмқҳ мқён„°нҺҳмқҙмҠӨк°„ мҶЎмҲҳмӢ мқҖ л„ӨнҠёмӣҢнҒ¬ л Ҳмқҙм–ҙ7мқҳ HTTP POST л©”м„ңл“ңмҷҖ мӣ№мҶҢмј“ н”„лЎңнҶ мҪңмқ„ нҷңмҡ©н•ңлӢӨ.

Rest API vs GraphQL

APIм—җ к°ҖмһҘ нқ”н•ҳкІҢ мӮ¬мҡ©лҗҳлҠ” кІғмқҙ REST APIмқҙлӢӨ. RESTлҠ” URL, METHODлҘј мЎ°н•©н•ҳм—¬ мӮ¬мҡ©н•ҳкё° л–„л¬ём—җ лӢӨм–‘н•ң Endpointк°Җ мЎҙмһ¬н•ңлӢӨ. л°ҳл©ҙ GQLмқҖ лӢЁ н•ҳлӮҳмқҳ Endpointк°Җ мЎҙмһ¬н•ңлӢӨ.

м¶ңмІҳ : https://blog.apollographql.com/graphql-vs-rest-5d425123e34b

RESTлҘј мӮ¬мҡ©н•  л•Ңмқҳ к°ҖмһҘ нҒ° л¬ём ңм җмқҖ Over fetchingмқҙлӢӨ. вҖҳFetchingвҖҷмқҖ к°Җм ёмҳӨлӢӨ лқјлҠ” лң»мқҙлӢӨ. RESTлҘј мӮ¬мҡ©н•  кІҪмҡ° API мһ‘м„ұмһҗк°Җ мһ‘м„ұн•ң лҚ°мқҙн„° кө¬мЎ°лҢҖлЎң лӘЁл“  м •ліҙлҘј л°ӣм•„мҳӨкІҢ лҗңлӢӨ. н•„мҡ”н•ң м •ліҙк°Җ ID н•ҳлӮҳмқҙлҚ”лқјлҸ„, USERм—җ лҢҖн•ң м •ліҙ к°қмІҙ лӘЁл‘җлҘј л°ӣм•„мҷҖм•ј н•ҳлҠ” кІғмқҙлӢӨ.

мқҙлҹ° л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ лӮҳмҳЁ кІғмқҙ GraphQLмқҙлӢӨ. GraphQLм—җм„ңлҠ” м–»кі мһҗн•ҳлҠ” м •ліҙлҘј лӘ…мӢңн•ҳм—¬ мӮ¬мҡ©н•ңлӢӨ.
м–»кі мһҗ н•ҳлҠ” мҶҚм„ұмқ„ лӘ…мӢңн•ҳм§Җ м•Ҡмңјл©ҙ мҳӨлҘҳк°Җ л°ңмғқн•ңлӢӨ.

н•өмӢ¬ мҡ”мҶҢ

GraphQLмқҖ мӮ¬мҡ©мһҗк°Җ м§Ғм ‘ мһ‘м„ұн•ң мҠӨнӮӨл§ҲлЎң лҸҷмһ‘н•ңлӢӨ. мҠӨнӮӨл§Ҳ нҢҢмқјмқҳ нҢҢмқјлӘ…мқҖ schema.graphqlлЎң мһ‘м„ұн•ҳлҠ” кІғмқҙ мқјл°ҳм ҒмқҙлӢӨ. мҠӨнӮӨл§Ҳ нҢҢмқјм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ мҡ”мҶҢл“Өмқҙ м„ м–ёлҗңлӢӨ.

  • Query

QueryлҠ” лҚ°мқҙн„°лІ мқҙмҠӨлЎңл¶Җн„° лҚ°мқҙн„°лҘј к°Җм ёмҳ¬ л•Ң мӮ¬мҡ©н•ңлӢӨ. мҝјлҰ¬мқҳ кІ°кіјлҠ” JSON нҳ•мӢқмңјлЎң л°ҳнҷҳлҗңлӢӨ.

  • Mutation

MutationмқҖ лҚ°мқҙн„°лІ мқҙмҠӨмқҳ лӮҙмҡ©мқ„ мҲҳм •н•  л•Ң мӮ¬мҡ©н•ңлӢӨ.

мҠӨнӮӨл§Ҳ нҢҢмқј мқҙмҷём—җ л°ҳнҷҳ лЎңм§Ғмқ„ мһ‘м„ұн•ҳлҠ” нҢҢмқјлЎң Resolverк°Җ мһҲлӢӨ. GraphQL м„ңлІ„лҠ” ResolverлҘј м°ҫм•„ QueryмҷҖ Mutationм—җ мқҙм—җ н•ҙлӢ№н•ҳлҠ” н•ЁмҲҳлҘј мӢӨн–үн•ңлӢӨ.
